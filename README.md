# خطوات — الخطة التسويقية | Khatawat Pitch Deck

عرض تقديمي تفاعلي احترافي للخطة التسويقية الأولية ومشروع منصة الأم والطفل.

A premium interactive pitch deck for Khatawat Agency's marketing plan for a mother-and-child e-commerce platform.

## Features

- 🎨 Premium dark theme (Deep Navy + Cyan/Gold accents)
- 📱 Fully responsive (desktop, tablet, mobile)
- ➡️ RTL (Right-to-Left) layout
- ⌨️ Keyboard navigation (arrow keys)
- 👆 Touch/swipe support
- 🔢 Animated number counters
- ✨ Glow animations and smooth transitions
- 🔗 Clickable glowing link buttons

## How to View Locally

Simply open `index.html` in any modern browser. No build step or server needed.

## 🚀 Deploy to Vercel (Step-by-Step Guide)

Follow these simple steps to deploy this presentation live on the internet:

### Step 1: Create a Vercel Account

1. Go to [vercel.com](https://vercel.com)
2. Click **"Sign Up"**
3. Choose **"Continue with GitHub"** and authorize Vercel to access your GitHub account

### Step 2: Import Your Repository

1. After signing in, you'll see the Vercel dashboard
2. Click the **"Add New..."** button → then **"Project"**
3. You'll see a list of your GitHub repositories
4. Find **`pitch-deck-presentation`** in the list and click **"Import"**

### Step 3: Configure & Deploy

1. On the configuration page:
   - **Framework Preset**: Select **"Other"** (this is a static site, no framework needed)
   - Leave all other settings as default
2. Click the **"Deploy"** button
3. Wait 30-60 seconds for the deployment to complete

### Step 4: Your Site is Live! 🎉

1. Vercel will show you a success screen with a preview of your site
2. Click **"Visit"** to see your live presentation
3. Your URL will be something like: `pitch-deck-presentation.vercel.app`
4. You can share this URL with anyone!

### Optional: Custom Domain

If you want a custom domain (e.g., `pitch.khatawat.com`):
1. Go to your project settings in Vercel
2. Click **"Domains"**
3. Add your custom domain and follow the DNS instructions
